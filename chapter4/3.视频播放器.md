

## 视频播放器

### 播放本地视频

注意：本地存在视频

```js
  Video({
        //视频源
        src: $r("app.media.mov_bbb"),
      })
```

### 播放网络视频

需要添加网络权限entry/src/main/module.json5

```js
   "requestPermissions": [
      {
        "name": "ohos.permission.INTERNET"
      }
    ],
```





```js
  Video({
        //视频源
        // src: $r("app.media.mov_bbb"),
        //封面
        src:"https://www.w3schools.com/html/movie.mp4",
      })
```



### 添加预览封面

```js
  Video({
        //视频源
        // src: $r("app.media.mov_bbb"),
        //封面
        src:"https://www.w3schools.com/html/movie.mp4",
        previewUri: $r("app.media.15")
      })
```
